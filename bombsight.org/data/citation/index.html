Citation Guide

Users who wish to cite material from the website in publications should, as with all internet publications, cite the URL (www.bombsight.org) and the date on which the website was consulted. Additionally, they should also cite the version number (1.0).

 

Citations of pages/blog posts from the other project pages should cite the title of the page/blog post, such as "Technical overview of the Android mobile application", using the following format:

 

  • "Technical overview of the Android mobile application ", Bomb Sight (www.bombsight.org version 1.0, 17 November 2012).
